# Librerias para PHP

路 AccesoDatos.php (Libreria para conectarse a una base de datos mysql)

路 LeerEmails.php (Libreria para leer emails de un servidor de correo)

路 Funciones.php (Libreria con algunas funciones utiles)

    - Crear una cadena aleatorea
    - Convierte <br> en nl

    - Saca el Mime-Content-Type de un fichero

路 WooCommerce_Curl.php (Libreria con funciones para conectarse a un WooCommerce mediante CURL)
